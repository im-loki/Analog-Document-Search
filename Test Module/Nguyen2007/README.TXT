The corpus consists of 211 scientific publications, 
each has 4 different formats: PDF, HTML, plain text, and XML. 
The documents come with author assigned and/or reader-assigned keyphrases.
There are 156 documents which has reader-assigned keyphrases

Each sub-folder contains following files:
	.pdf : document in PDF format
	.html: document in HTML format
	.txt : document in plain text
	.xml : document in XML format, each document is presented with their logical sections
	.kwd : author-assigned keyphrases
	.term: general terms
	.acm : categories and subject descriptors
	KEY/ID.key  : reader assigned keyphrases, each file contains a keyphrase set 
				  annotated by user whose id is ID
